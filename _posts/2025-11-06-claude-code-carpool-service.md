## 📝 快速配置指南

### 方式一：Claude Code 桌面版

1. 打开 Claude Code 设置
2. 找到 API 配置选项
3. 输入我们提供的调用地址和 key
4. 保存配置，即可开始使用

### 方式二：API 调用

```bash
# 设置环境变量
export CLAUDE_API_BASE="你的调用地址"
export CLAUDE_API_KEY="你的调用key"

# 开始使用
claude-code --api-base=$CLAUDE_API_BASE --api-key=$CLAUDE_API_KEY
```
